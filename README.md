# Shopping list

## Setup

This project uses nix. Nix is awesome - https://nixos.org/guides/how-nix-works/

Clone the repo, and then run:

`nix develop`

This will install flutter, android sdk and everything else you need to get going. Once that's done, ensure you have the flutter addin in your vscode, hit F5 or Run -> start debugging. You should see the app start up.

## Goals

1. For me to learn a bit about how flutter and the app-building process works. I've always wanted to actually publish an app but found Java too....Java like. Since I heard about flutter I was always intrigued by it. Now I'm finally making something.
2. To actually make a useful app. It will be 100% free, and open source I guess, at least at this stage. Focus on no ads, no in-app purchases and no data gathering - donationware likely.
